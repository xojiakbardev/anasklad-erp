# frontend/

Anasklad-ERP admin UI — React 19 + Vite 6 + TanStack (Router/Query/Table) + Tailwind v4 + Framer Motion.

Design system: "Terminal of the Silk Road" — see [`../DESIGN.md`](../DESIGN.md).

## Dev

```bash
pnpm install           # or npm/yarn
pnpm dev               # http://localhost:5173
```

The dev server proxies `/api/*` to `http://localhost:8000` (see `vite.config.ts`).

## Struktura

```
src/
├── api/              # HTTP клиентлар (ky + handle())
├── components/       # умумий UI элементлар
│   ├── AppShell     # Sidebar + Outlet + status bar
│   ├── IkatStripe   # signature ornament element
│   ├── Logo         # brand mark
│   └── Panel        # flat card with optional ikat stripe
├── i18n.ts          # uz/ru resurslar
├── lib/
│   ├── api.ts       # ky instance + ApiError + 401 refresh
│   ├── cn.ts        # tailwind-merge helper
│   └── format.ts    # ledger-style number formatting
├── routes/          # TanStack file-based routing
│   ├── __root.tsx
│   ├── login.tsx
│   ├── _auth.tsx           # authed layout
│   ├── _auth.index.tsx     # /  (dashboard)
│   ├── _auth.integrations.tsx
│   └── _auth.{products,orders,stocks,finance}.tsx
├── stores/
│   └── auth.ts      # Zustand + persist + refresh flow
├── styles/
│   └── index.css    # design tokens + utility classes
└── main.tsx
```

`src/routeTree.gen.ts` is generated by the TanStack Router Vite plugin on first dev run.
